# Backend Diagrams

This folder contains Mermaid diagrams generated from the current backend implementation.

## Files
- `architecture.md` - High-level backend architecture and component interactions.
- `sequence-oauth-login.md` - OAuth login flow and JWT issuance.
- `sequence-api-me.md` - Authenticated profile fetch flow for `/api/me`.
- `sequence-document-upload.md` - End-to-end document upload flow to GCS.
- `class-domain.md` - Domain entities and relationships.
- `class-application.md` - Use case classes and dependencies.
- `class-security-auth.md` - Security/authentication components.
- `class-repositories-storage.md` - Repository and storage interfaces/implementations.

## How to view
- Open these markdown files in Cursor or GitHub to render Mermaid diagrams.
- If rendering is disabled, copy the Mermaid block into any Mermaid live editor.
